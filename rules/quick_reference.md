2. **현재 상태**: "현재 FAISS 인덱스 사용"
3. **구체적 요구사항**: "메모리 사용량 최적화"
4. **제약사항**: "정확도 유지"
5. **성공 기준**: "검색 속도 2배 향상"

### 커스터마이징 방법
- 기술 스택 변경: "FAISS" → "Pinecone"
- 성능 요구사항: "50%" → "2배"
- 도메인 특화: "일반" → "법률/의료"
- 사용자 그룹: "개발자" → "엔드유저"
